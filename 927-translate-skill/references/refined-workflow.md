# Refined 翻译工作流

出版级翻译的完整步骤。每步产出保存为独立文件。

## Step 1: 内容分析 → `01-analysis.md`

深度分析原文，产出分析文档。参照 SKILL.md 中的分析模板。

### 1.1 比喻/习语映射（重点）

识别原文中所有比喻、明喻、习语和修辞表达。对每一个：

1. **原文**：原始短语
2. **含义**：作者实际想表达什么
3. **直译风险**：逐字翻译是否不自然、丢失含义、或让目标读者困惑？
4. **处理策略**：
   - **意译**：丢弃源语言意象，用目标语言直接表达含义
   - **替换**：用目标语言中同等效果的习语/比喻替换
   - **保留**：原始意象在目标语言中同样有效时保留

### 1.2 情感色彩标注

标注原文中携带主观情感的用词（如 "alarming"、"haunting"、"striking"），确保翻译时保留同等情感回响，而不是扁平化为中性描述。

### 1.3 读者理解难点

根据目标受众校准：
- **领域术语**：缺乏公认翻译或直译无意义的术语
- **文化典故**：源文化特有的习语、历史事件、流行文化
- **隐含知识**：作者假设但目标读者可能缺少的背景
- **双关/文字游戏**：跨语言无法移植的表达

## Step 2: 初稿 → `03-draft.md`

基于分析结果翻译全文：
- 严格使用术语表
- 按比喻映射表执行处理策略
- 为理解难点添加译注
- 匹配分析确定的语气和文体

## Step 3: 审校 → `04-critique.md`（仅诊断，不修改）

逐段对照原文审查初稿，产出诊断报告：

### 3.1 准确性 & 完整性
- 逐段对照，检查事实、数据、专有名词
- 标记添加、遗漏或篡改的内容
- 检查术语一致性

### 3.2 欧化语言检测（CJK 目标语言）

| 问题类型 | 症状 | 示例修正 |
|----------|------|----------|
| 多余连接词 | 因此/然而/此外 过度使用 | 上下文已暗示时删除 |
| 被动语态滥用 | 不必要的 被/由/受到 | 改为主动语态 |
| 名词堆砌 | 长修饰语链 | 拆分为多个短句 |
| 生硬"是...的" | 从英语 "It is...that" 硬搬 | 改为自然表达 |
| 过度名词化 | "进行了讨论" | 改为 "讨论了" |
| 多余代词 | 不必要的 他/她/它/我们 | 中文中省略 |

### 3.3 比喻/习语 & 情感保真
- 对照分析阶段的比喻映射：是否按策略执行？
- 标记仍被直译的比喻
- 检查情感词是否被扁平化

### 3.4 表达 & 逻辑
- 标记"翻译腔"严重的句子
- 检查段落间的逻辑衔接
- 标注需要句式重组的位置

### 3.5 译注质量
- 注释是否准确、简洁、有帮助？
- 是否遗漏了需要注释的难点？
- 是否过度注释了显而易见的术语？

诊断报告格式：
```markdown
## 准确性问题
- [位置]: [描述]

## 欧化语言
- [类型]: [初稿原文] → [修正建议]

## 比喻/情感
- [原文] → [初稿处理] → [建议改进]

## 表达问题
- [位置]: [问题] → [建议]

## 译注
- [添加/删除/修改]: [术语] — [原因]

## 总结
[整体评估：X 个关键问题，Y 个改进建议]
```

## Step 4: 修订 → `05-revision.md`

读取初稿 + 审校报告，逐项修复：
- 修正所有准确性问题
- 消除欧化语言
- 重新处理直译的比喻
- 恢复被扁平化的情感
- 重组生硬句式
- 调整译注

## Step 5: 润色 → `translation.md`

最终通读 `05-revision.md`：
- 作为独立文章阅读——是否像目标语言原创？
- 段落衔接是否流畅？
- 叙事语调是否全篇一致？
- 最终扫描：是否还有"翻译感"的比喻或扁平化的情感？
- 术语全文一致性最终检查
- 格式验证（标题、加粗、链接、代码块）

## 长文分块 + Refined

当内容超过分块阈值且使用 refined 模式：

1. 先对**完整文档**做分析（Step 1）
2. 分块翻译初稿，每块共享分析结果
3. 合并所有块 → `03-draft.md`
4. 对合并稿做完整审校 → `04-critique.md`
5. 全文修订 → `05-revision.md`
6. 全文润色 → `translation.md`
7. **跨块一致性检查**：术语、语调、衔接
