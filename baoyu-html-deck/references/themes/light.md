# Light 浅色主题

## 配色

```css
--bg-primary: #ffffff;
--bg-secondary: #f5f5f5;
--bg-card: #ffffff;
--text-primary: #1a1a1a;
--text-secondary: #666666;
--accent: #0066cc;
--accent-secondary: #00cc66;
--accent-gradient: linear-gradient(135deg, #0066cc, #00cc66);
--border-color: rgba(0, 102, 204, 0.2);
--shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
```

## 特点

- 白色背景，干净清爽
- 蓝色+绿色强调色
- 清晰的阴影效果
- 适合明亮环境或打印
