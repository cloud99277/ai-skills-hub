# Dark 暗黑主题

## 配色

```css
--bg-primary: #0d0d0d;
--bg-secondary: #1a1a1a;
--bg-card: #252525;
--text-primary: #ffffff;
--text-secondary: #888888;
--accent: #ff6b6b;
--accent-secondary: #feca57;
--accent-gradient: linear-gradient(135deg, #ff6b6b, #feca57);
--border-color: rgba(255, 107, 107, 0.3);
--shadow: 0 8px 32px rgba(255, 107, 107, 0.15);
```

## 特点

- 纯黑背景，极简风格
- 珊瑚红+金菊黄强调色
- 柔和发光效果
- 适合暗光环境演示
