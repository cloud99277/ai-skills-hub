# Blueprint 科技蓝主题

## 配色

```css
--bg-primary: #1a1a2e;
--bg-secondary: #16213e;
--bg-card: #0f3460;
--text-primary: #ffffff;
--text-secondary: #a0a0a0;
--accent: #00d9ff;
--accent-secondary: #00ff88;
--accent-gradient: linear-gradient(135deg, #00d9ff, #00ff88);
--border-color: rgba(0, 217, 255, 0.3);
--shadow: 0 8px 32px rgba(0, 217, 255, 0.2);
```

## 特点

- 深色背景，突出科技感
- 青色+绿色渐变作为强调色
- 发光效果和边框
- 网格背景图案
