# 布局类型说明

## 封面布局

- 自动识别第一页为封面
- 大标题 + 副标题居中
- 特殊背景效果

## 默认布局

- `# 标题` 开头
- 下方为普通段落内容
- 标题与内容有明显间距

## 卡片布局

- `## 标题` 开头
- 跟随无序列表 (`-`)
- 每两个列表项为一组：标题 + 内容
- 显示为网格卡片

示例：
```markdown
## 功能特点
- 卡片1标题
  卡片1内容
- 卡片2标题
  卡片2内容
```

## 列表布局

- `## 标题` 开头
- 跟随无序列表
- 列表项居中显示
- 适合展示要点

## 步骤布局

- 数字列表 (`1.`, `2.`, `3.`)
- 水平排列步骤
- 带序号标识
